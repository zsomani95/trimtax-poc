Property Data — Core Revenue Engine
Property Search
GET /api/properties/search
Query: ?q=partial_address (min 3 chars)
SQL: WHERE site_addr_1 ILIKE '%q%' LIMIT 10
Uses idx_properties_addr B-tree index
Returns: acct, address, city, zip, county, owner, value, sqft
1.5M row tabledebounced 300ms
Savings Estimate
GET /api/properties/[acct]/estimate
1. Fetch subject property
2. Query neighborhood comps (same neighborhood_code, ±30% sqft)
3. Fallback to zip if <5 comps
4. Compute $/sqft per comp
5. Winsorize at 10th/90th percentile
6. Median $/sqft × subject sqft = arguedValue
7. Confidence: comp count + CV + overage%
§41.43Winsorizationlow/med/high confidence

  1. Re-download real_acct.txt from HCAD bulk files

  4. Run node scripts/import-to-neon.js
Fort Bend County (Jan each year)
  1. Download latest Property Data Export from fbcad.org/data-files/
  2. Extract Property.zip + Owner.zip
  3. Update file paths in filter-fbcad.js
  4. Run node scripts/filter-fbcad.js
  5. Run node scripts/import-fbcad-to-neon.js
