Key API Calls Per Screen
GET /api/properties/search
Used by: Intake Step 1
Query: ?q=partial_address
Returns: up to 10 matching properties
Index: idx_properties_addr
GET /api/properties/[acct]/estimate
Used by: Intake Step 2
§41.43 comp engine
Winsorized median $/sqft
Returns: savings range + confidence
POST /api/submissions
Used by: Intake Step 4
Creates submission row, status=pending
Redirects to /analysis/[id]
POST /api/generate-pdf
Used by: Sign Step 2
Forms 50-132 + 50-162 via pdf-lib
Returns: PDF download blob
PATCH /api/submissions/[id]
Used by: Sign Step 3
Stores signature_image (base64 PNG)
Sets status=signed, signedAt=now
GET /api/tracker/[id]
Used by: Tracker page
Returns: submission + all notes
Requires Bearer token header
